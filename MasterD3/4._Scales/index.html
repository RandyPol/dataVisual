Quantile Scale

The quantile scale will evenly distribute the numbers across all buckets. It will divide the number of items in the domain by the number of items in the output range. So 100 divided by 3. It will determine that each bucket can represent 33 values each. 100 divided by 3 will result in a decimal value. If the domain cannot be evenly divided, it will put the remainder into the final bucket.
